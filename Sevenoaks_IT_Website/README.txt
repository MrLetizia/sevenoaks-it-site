Sevenoaks IT Subjects Website

To use:
1. Add your PDFs and images to the 'assets' folder.
2. Open index.html in a browser to preview locally.

To publish on GitHub Pages:
1. Create a GitHub account and a new repository (e.g., sevenoaks-it-site).
2. Upload the files in this folder.
3. Go to Repository Settings > Pages.
4. Set source to 'main' branch and root folder.
5. Visit your published link (e.g., https://yourusername.github.io/sevenoaks-it-site/).
